# Rainfall Prediction System

A Machine Learning based web application that predicts rainfall using climate parameters.

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## Machine Learning Model
Linear Regression

## Features
- Select State
- Select Month
- Input Temperature
- Input Humidity
- Input Wind Speed
- Predict Rainfall instantly

## How to Run the Project

1. Clone the repository

git clone https://github.com/BhagirathArgade/Rainfall_Prediction_System.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py

## Dataset
Synthetic climate dataset generated using Python for demonstration purposes.
